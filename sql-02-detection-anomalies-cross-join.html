Repérer les magasins qui vendent anormalement peu

Compare la performance de chaque magasin à la moyenne de tous les magasins, pour repérer instantanément ceux qui décrochent significativement, via un produit croisé (CROSS JOIN).

1

CA moyen par magasin

On calcule le chiffre d'affaires moyen de chaque magasin, groupé par groupe et par enseigne.

2

Moyenne du marché

On calcule ensuite la moyenne globale, tous magasins confondus.

3

Comparaison en une passe

Un CROSS JOIN permet de comparer chaque magasin à cette moyenne globale en une seule fois.

4

Seuil d'alerte

On ne garde que les magasins qui vendent moins de 50 % de la moyenne du marché.

-- ============================================================================
-- 900.care · Détection d'anomalies structurelles (CROSS JOIN)
-- ============================================================================
-- Cartographie la santé financière d'un réseau de distribution en confrontant
-- instantanément la performance historique de chaque magasin à une constante
-- de marché calculée dynamiquement, via un produit croisé (CROSS JOIN).
-- ============================================================================

/* ①  CA moyen par magasin */
WITH store_performance AS (
  SELECT
    sales.store_name AS store_name,
    sales.retail_group AS retail_group,
    sales.banner_name AS banner_name,
    ROUND(AVG(sales.amount_ht), 2) AS avg_revenue_store
  FROM `analytics_warehouse.fct_sell_out` AS sales
  WHERE COALESCE(sales.is_test_row, FALSE) = FALSE
  GROUP BY 1, 2, 3
),
/* ②  Moyenne globale de marché, calculée à partir du résultat précédent */
global_market_average AS (
  SELECT
    AVG(avg_revenue_store) AS global_avg_revenue
  FROM store_performance
)
SELECT
  store.store_name,
  store.retail_group,
  store.banner_name,
  store.avg_revenue_store AS store_average_revenue,
  ROUND(market.global_avg_revenue / 2, 2) AS critical_alert_threshold
FROM store_performance AS store
/* ③  On compare chaque magasin à la moyenne globale en une seule passe */
CROSS JOIN global_market_average AS market
/* ④  Seuil d'alerte : magasins sous 50 % de la moyenne, CA positif */
WHERE store.avg_revenue_store < (market.global_avg_revenue / 2)
  AND store.avg_revenue_store > 0
ORDER BY store.avg_revenue_store ASC;
