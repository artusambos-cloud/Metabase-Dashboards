# Metabase Dashboards — 900.care Retail Analytics (exemples anonymisés)

Ce repo présente une sélection de dashboards Metabase construits pour le suivi retail (ventes, commandes, visites terrain, commissions agences, rétention client). Il s'agit de reconstructions visuelles des vrais dashboards : mêmes types de graphiques, même structure, mais avec des données entièrement fictives — pensées pour être partagées publiquement sans exposer de chiffres d'affaires, de partenaires ou de collaborateurs réels.

## Comment consulter les dashboards

Chaque fichier `.html` est autonome : télécharge-le et ouvre-le directement dans un navigateur, ou active [GitHub Pages](https://docs.github.com/en/pages/quickstart) sur ce repo pour les consulter en ligne sans rien télécharger.

## À propos des données

Tous les chiffres (CA, volumes, taux, montants, dates précises) sont générés artificiellement. Les noms d'agents commerciaux, d'agences partenaires et de magasins individuels sont fictifs.

Dashboards originaux construits sous [Metabase](https://www.metabase.com/). Les reconstructions de ce repo utilisent [Chart.js](https://www.chartjs.org/)
